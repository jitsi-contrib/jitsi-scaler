Jitsi-Scaler Chart Repository

This page serves as the repository base for the Jitsi-Scaler charts.

For Helm Users:

To use this repository, add it to your Helm client:

helm repo add jitsi-scaler https://jitsi-contrib.github.io/jitsi-scaler/

For Documentation:

Please visit the main GitHub repository: Jitsi-Scaler GitHub Repository

The chart catalog is available at index.yaml
